# HR Analytics
***
- Leveraged the latest functionalities such as card visuals and slicers to enhance data visualization and interactivity within Power BI dashboards.
- Used figma to create the visually appealing dashboard background and used the same
- Leveraged **dynamic data labels** in Power BI for enhanced interactivity and dynamic data exploration, empowering informed decision-making.

## Dashboard is divided in multiple pages / reports

1. People service report / page
    -  This report highlights overall headcounts in the organization
    -  These are further divided into department, gender, business unit, age range, Marital Status
    -  This report also focuses on New hires and headcount trend in last 12 months
     
<img align="center" width="700" src="https://github.com/amitgajkal/PowerBI-People-Service-Dashboard/blob/main/Resources/1%20people%20service%20dashboard.png" />

2. Termination Analysis
  - This report highlights termination of employees by
     -  gender,
     -  voluntary termination,
     -  Involuntary termination,
     -  termination tenure,
     -  Termination by age
  <img align="center" width="700" src="https://github.com/amitgajkal/PowerBI-People-Service-Dashboard/blob/main/Resources/2%20termination%20analysis.png" />

3. Training Report
  - Report includes details of training report
    - Training cost per employee
    - Total Training cost
    - Training duration
    - Number of training programs
    - Number of participants
    - Training cost per month
    - Training duration per month
    - Training program cost
    - Program completion status
    - Total programs by training type
  <img align="center" width="700" src="https://github.com/amitgajkal/PowerBI-People-Service-Dashboard/blob/main/Resources/3%20Training%20report.png" />


4.  Recruitment Report
  - It gives brief report on New recruitment / New hires
      - Training cost per employee
      - Vacancies filled, active vacancies
      - Number of Applicants
      - Candidate assessment rejected, screeing, Hires
      - Candidate monthly hiring
      - Department wise hiring
        
  <img align="center" width="700" src="https://github.com/amitgajkal/PowerBI-People-Service-Dashboard/blob/main/Resources/4%20recruitment%20report.png"/>

  5.  Performance Report
  - It gives brief report on individual performances
      - individuals with different ratings
      - Performance with generation
      - Performance by race
      - Performance by Business Unit
      - Performance by gender
      - Performance by age group
        
  <img align="center" width="700" src="https://github.com/amitgajkal/PowerBI-People-Service-Dashboard/blob/main/Resources/5%20performance%20report.png"/>
     
  6.  Employee Profile
  - It gives brief report on individual details
      - Focuses on least performance rating
      - Employee personal and professional details
      - Leaves information
      - Performance matrix
      - Training details

  <img align="center" width="700" src="https://github.com/amitgajkal/PowerBI-People-Service-Dashboard/blob/main/Resources/6%20employee%20profile.png"/>
